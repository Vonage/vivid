# vwc-relative-time

Show relative time to `datetime`

## Properties

| Property                  | Modifiers | Type                                      | Description                                      |
|---------------------------|-----------|-------------------------------------------|--------------------------------------------------|
| `live`                    |           | `boolean`                                 | When set to `true`, the component will automatically update |
| `datetime`                |           | `integer`                                 |  The target timestamp in milliseconds since epoch |
