### Installation

```bash
yarn add vwc-scheme-loader
```

place module reference in the head of your html to apply vivid styling for your application and to the vivid web components
either by a plain old script tag (with no defer/async).
or by the following 'modulepreload' approach where supported
our goal is to make sure the script gets high priority

```
<script rel="modulepreload" src="../node_modules/@vonage/scheme-loader/scheme-loader.js"></script>
```

preloading the scheme is important as it is a critical styling resource

```
<script type="module" src="../node_modules/@vonage/scheme-loader/scheme-loader.js"></script>
```
