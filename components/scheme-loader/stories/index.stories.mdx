## How to use

### Installation

```bash
yarn add vwc-scheme-loader
```

place module reference in the head of your html to apply relevant styling for your application and the vivid web components

```
<script rel="modulepreload" src="../node_modules/@vonage/scheme-loader/scheme-loader.js"></script>
```

preloading the scheme is important as it is a critical styling resource

```
<script type="module" src="../node_modules/@vonage/scheme-loader/scheme-loader.js"></script>
```
