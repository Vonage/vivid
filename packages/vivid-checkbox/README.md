# \<vivid-checkbox>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i vivid-checkbox
```

## Usage
```html

  import 'vivid-checkbox/vivid-checkbox.js';
</script>

<vivid-checkbox></vivid-checkbox>
```
