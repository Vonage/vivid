# vwc-audio

Basic audio player

## Properties

| Property      | Type     | Description                     |
|---------------|----------|---------------------------------|
| `currentTime` | `number` | Gets/Sets the playhead position |
| `src`         | `string` | Gets/Sets the audio source      |

## Methods

| Method  | Type       | Description     |
|---------|------------|-----------------|
| `pause` | `(): void` | Pauses playback |
| `play`  | `(): void` | Starts playback |
